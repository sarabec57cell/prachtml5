¿Qué es HTML5?

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web.

Estructura basica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi pagina</title> </head> <body> </body> </html>

doctype.Le indica al navegador qué versión de HTML se está usando, head.Contiene información sobre la página, pero no el contenido visible principal. body.Aquí va todo el contenido visible de la página web.

Etiquetas principales

h1 - h6

Título más importante (principal)

p

Define un párrafo de texto.

img

Sirve para mostrar imágenes,no tiene etiqueta de cierre.

a

Define un enlace (link) para ir a otra página o recurso.

button

Se usa para ejecutar acciones (por ejemplo, enviar formularios)

ul / ol / li

ul:Muestra una lista con viñetas (puntos),ol:Muestra una lista numerada, li:Elemento de lista.Se usa dentro de ul o ol para cada elemento.

HTML5 semantico

header

Es la parte superior de una página o sección.

nav

Se usa para menús de navegación.

main

Representa el contenido principal de la página.

section

Define una sección temática dentro de la página.

article

footer

Es el pie de página,suele contener:Información de contacto

Formularios en HTML5

Los formularios en HTML se usan para recoger información del usuario, como datos de contacto, búsquedas, registros, comentarios, etc.Sus etiquetas principales son: form Contenedor del formulario label:Texto descriptivo del campo input:Campo de entrada textarea:Texto largo select:Lista desplegable button:Botón (enviar o acción)










Multimedia en HTML5

HTML5 introdujo muchas mejoras para manejar contenido multimedia (imágenes, audio y video) sin necesidad de plugins externos como Flash.

Imagenes

Se usa para insertar imágenes en una página web. Funciones importantes: src: ruta de la imagen alt: texto alternativo (clave para accesibilidad)

Audio

Permite reproducir sonido directamente en la página web.

Video

La etiqueta video se utiliza en HTML5 para insertar videos directamente en una página web

Accesibilidad

El atributo alt es esencial para accesibilidad, usabilidad y SEO en imágenes..

Conclusión

HTML5 es importante porque mejora la estructura del sitio con etiquetas semánticas, facilita la multimedia y la accesibilidad, que hacen el desarrollo web más completo.
